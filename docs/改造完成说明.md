## 📱 移动端适配要点

### 关键媒体查询
```css
@media screen and (max-width: 768px) {
  /* 移动端样式 */
}
```

### 单位转换规则
- **字体**: `clamp(14px, 2.5vw, 16px)` - 流式字体
- **间距**: `rem` - 相对根字体大小
- **容器**: `100%` / `vw` - 相对视口宽度
- **按钮**: `min-height: 44px` - 符合触控规范

### 布局策略
- **头部**: PC横向 → 移动端纵向
- **内容**: PC双栏 → 移动端单栏
- **配置页**: PC左右分栏 → 移动端上下布局
